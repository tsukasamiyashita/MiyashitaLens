MiyashitaLens v1.5.0

MiyashitaLens は、画面上のテキストを切り取って、Gemini API を使用して翻訳や辞書検索を行うデスクトップアプリです。
直感的な操作で、画像内のテキストを瞬時に抽出し、AIによる高度な解析結果を得ることができます。

主な特徴

高精度なテキスト抽出: 画面の範囲指定により、必要な部分だけを素早く切り取れます。Windows標準のOCR機能（WinRT）を活用し、高速かつ正確にテキストを読み取ります。

多機能な処理モード:

日本語翻訳: 抽出したテキストを自然な日本語に翻訳します。

英語翻訳: 抽出したテキストを自然な英語に翻訳します。

辞書モード: 語句の意味、発音、使い方を簡潔に解説します。

コピーモード: 画像から抽出したテキストを即座にクリップボードへ直接コピーします。

履歴保存機能: 過去に処理したテキストを最大100件まで自動保存します。固定（ピン留め）機能や、不要な履歴のクリア機能も備えており、効率的な管理が可能です。

柔軟なAPI詳細設定:

モデル選択: Gemini 1.5 Flash/Pro など、用途に合わせてモデルを選択可能です。

パラメータ調整: 温度（Temperature）や最大トークン数を調整し、出力の精度や長さを制御できます。

カスタムプロンプト: 独自の指示を追加することで、より目的に沿った回答を得ることができます。

プライバシーと効率: 課金枠（Paid Tier）を利用することで、入力データの学習利用を回避し、より高いレート制限で安定した利用が可能です。

使い方

起動: アプリを起動します。タスクトレイに常駐するため、いつでも素早く呼び出せます。

モード選択: メイン画面で「日本語翻訳」「英語翻訳」「辞書」「コピー」から目的のモードを選択します。

範囲指定: 「🔍 画面を切り取って実行」ボタンを押すと、画面が暗転します。マウスで翻訳したいテキストの範囲をドラッグして指定してください。

結果確認: 自動的にGemini APIが呼び出され、結果ウィンドウに処理結果が表示されます（コピーモードの場合は直接クリップボードに保存されます）。

処理結果は編集可能で、必要に応じて修正できます。

処理中にモードを切り替えて再解析することも可能です。
