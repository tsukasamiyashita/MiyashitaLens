MiyashitaLens v1.3.1

画面上の一部をマウスで囲むだけで、自動的に文字を読み取り、瞬時に翻訳や意味の検索ができるデスクトップアプリケーションです。
スマートフォンでおなじみの「Googleレンズ」のような機能を、Windows PCの画面上で簡単に利用することができます。

🌟 主な機能

ワンタッチ画面キャプチャ: アプリを起動して「画面を切り取って実行」を押し、マウスをドラッグするだけで直感的に範囲を指定できます。

3つの便利なモード: 「日本語翻訳」「英語翻訳」「辞書（意味と使い方）」の3つのモードを搭載し、目的に合わせていつでも切り替え可能です。

超高速な自動文字認識 (OCR): Windows標準の内蔵OCRエンジンを利用し、指定した範囲内の画像からテキストをPC内で瞬時に読み取ります。（インターネットへの画像アップロードが不要なため爆速です）

リアルタイムAI処理: 高速なAI（Gemini）を利用し、読み取った文章を指定のモードに合わせて即座に処理し、専用のポップアップウィンドウに表示します。

💻 動作環境

OS: Windows 11 / Windows 10

ネットワーク: インターネット接続（AI機能を利用するために必要です）

🚀 インストールと実行方法

配布された MiyashitaLens.exe ファイルを、デスクトップなどの分かりやすい場所に保存します。

MiyashitaLens.exe をダブルクリックして起動します。

初回起動時は、設定の読み込みに数秒かかる場合があります。

⚠️ Windows Defenderの青い警告画面が出た場合

初めてアプリを起動した際、Windowsのセキュリティ機能（SmartScreen）によって「Windows によって PC が保護されました」という青い警告画面が表示されることがあります。これはインターネットからダウンロードした新しいアプリに対して表示される標準的な画面です。

以下の手順で安全に起動してください：

メッセージ内の左側にある 「詳細情報」 という文字をクリックします。

右下に 「実行」 ボタンが現れるので、それをクリックします。

※アプリが起動します（次回以降はこの画面は表示されません）。

📖 基本的な使い方

アプリを起動し、メイン画面で使いたいモード（日本語翻訳・英語翻訳・辞書）を選択します。

「⚙️ API詳細設定」 から、ご自身のGemini APIキーを設定します（初回のみ）。

「🔍 画面を切り取って実行」 ボタンをクリックします。

画面全体が少し暗くなり、マウスカーソルが十字（＋）に変わります。

読み取りたい文章や画像の部分を、マウスの左ボタンを押したままドラッグして四角く囲みます。

マウスのボタンから指を離すと、即座に結果ウィンドウが表示されます。

上段に「読み取った元のテキスト」、下段に「AIによる処理結果」が表示されます。結果のテキストは自由にコピーして他の用途に使うことができます。

別のモードでの結果が見たい場合は、結果ウィンドウ上の「モード切替」ボタンを押すことで瞬時に再処理が可能です。
