Glans_Miya

画面上の一部をマウスで囲むだけで、自動的に文字を読み取り、瞬時に日本語へ翻訳するデスクトップアプリケーションです。
スマートフォンでおなじみの「Googleレンズ」のような機能を、Windows PCの画面上で簡単に利用することができます。

🌟 主な機能

ワンタッチ画面キャプチャ: アプリを起動すると画面が暗くなり、マウスをドラッグするだけで直感的に範囲を指定できます。

自動文字認識 (OCR): 指定した範囲内の画像から、日本語や英語のテキストを自動的に読み取ります。

リアルタイム翻訳: 読み取った文章を即座に日本語に翻訳し、専用のポップアップウィンドウに表示します。

💻 動作環境

OS: Windows 11 / Windows 10

インターネット接続（翻訳機能を利用するために必要です）

🚀 インストールと実行方法

配布された Glans_Miya.exe ファイルを、デスクトップなどの分かりやすい場所に保存します。

Glans_Miya.exe をダブルクリックして起動します。

初回起動時は、設定の読み込みに数秒かかる場合があります。

⚠️ Windows Defenderの青い警告画面が出た場合

初めてアプリを起動した際、Windowsのセキュリティ機能（SmartScreen）によって「Windows によって PC が保護されました」という青い警告画面が表示されることがあります。これはインターネットからダウンロードした新しいアプリに対して表示される標準的な画面です。

以下の手順で起動してください：

メッセージ内の左側にある 「詳細情報」 という文字をクリックします。

右下に 「実行」 ボタンが現れるので、それをクリックします。

アプリが安全に起動します（次回以降はこの画面は表示されません）。

📖 基本的な使い方

アプリ（Glans_Miya.exe）を起動します。

画面全体が少し暗くなり、マウスカーソルが十字（＋）に変わります。

翻訳したい文章や画像の部分を、マウスの左ボタンを押したままドラッグして四角く囲みます。

マウスのボタンから指を離すと、数秒後に結果ウィンドウが表示されます。

上段に「読み取った元のテキスト」、下段に「日本語への翻訳結果」が表示されます。結果のテキストは自由にコピーして他の用途に使うことができます。

確認が終わったら「閉じる」ボタンでウィンドウを終了します。
